# GPTs FORGE — GitHub Pages 公開セット

このフォルダは、`index.html` をそのまま GitHub Pages で公開するための最小構成です。

## 含まれているファイル
- `index.html`  
  メインのアプリ本体です。リポジトリのルートに置いてください。
- `.nojekyll`  
  GitHub Pages で静的サイトとしてそのまま公開しやすくするための補助ファイルです。

## いちばん簡単な公開手順
1. GitHubで新しいリポジトリを作成
2. このフォルダ内の `index.html` と `.nojekyll` をアップロード
3. リポジトリの **Settings** を開く
4. **Pages** を開く
5. 公開元のブランチを `main`、フォルダを `/ (root)` に設定
6. 保存して数十秒〜数分待つ
7. 発行されたURLにアクセスして動作確認

## 更新方法
- デザインやコードを直したい場合は、`index.html` を差し替えて再アップロードすればOKです。

## 注意点
- Gemini APIキーはユーザー自身が画面上で入力する仕様です。
- フロントエンド単体のため、APIキーをHTML内に直接書き込まないでください。
- 初回公開直後は、反映まで少し時間がかかることがあります。

## ローカル確認
ブラウザで `index.html` を直接開いて見た目確認はできます。
ただし、実運用は GitHub Pages 上で最終確認するのがおすすめです。
