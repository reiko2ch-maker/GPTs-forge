# GPTs FORGE — GitHub Pages 公開セット v2

より安定性を上げた版です。見た目は基本維持しつつ、内部は以下を改善しています。

- 生成をさらに細かいフェーズへ分割
- フェーズごとの自動リトライ
- 途中フェーズのチェックポイント保存
- 同じ入力で再実行した際の自動再開
- 履歴保存の IndexedDB 優先 + localStorage フォールバック
- 入力テキストの圧縮でリクエスト負荷を軽減

## 使い方
1. GitHubで新しいリポジトリを作成
2. このフォルダ内の `index.html` と `.nojekyll` をアップロード
3. Settings → Pages を開く
4. Branch を `main`、フォルダを `/ (root)` に設定
5. 数十秒〜数分待って公開URLへアクセス

## 注意
- APIキーは画面上でユーザーが入力する仕様です
- HTML内にキーを埋め込まないでください
- 同じ入力で再実行すると、保存済みフェーズから自動再開します
